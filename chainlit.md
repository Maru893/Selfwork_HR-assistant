# HR Assistant

Benvenuto in **HR Assistant**.

Questa applicazione aiuta a cercare candidati e informazioni dentro CV e documenti indicizzati.

Puoi usarla per:

* trovare candidati con competenze specifiche
* confrontare profili in base a una richiesta
* cercare esperienze, tecnologie o responsabilità nei CV
* caricare nuovi documenti direttamente dall'interfaccia

Formati supportati:

`.txt`, `.pdf`, `.doc`, `.docx`, `.csv`, `.json`, `.xlsx`, `.pptx`, `.html`, `.xml`, `.zip`

Il sistema usa un approccio **RAG**: prima recupera le parti più rilevanti dai documenti, poi genera una risposta chiara usando il modello LLM configurato.
